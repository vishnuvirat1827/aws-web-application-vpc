# aws-web-application-vpc
# Deploying a Web Application within the VPC

## Objective
Deploy a simple web application within an AWS VPC using an EC2 instance and a web server.

## Topics Covered
- AWS EC2
- AWS VPC
- Apache/Nginx
- Web Application Deployment
- Security Groups

## Outcome
Learned how to deploy a web application on an EC2 instance within a VPC and configure network access securely.
